# ChessSimulator
Ceci est un jeu d'échecs codé en Python (non terminé)  
Fonctionnalités (très basiques) :
  - Calcul des coups légaux
  - Echecs et Echecs et mat
  - Roque
  - Prise en passant
  - UI réalisée avec Pygame

Bugs connus :
  - Certains coups sont légaux alors qu'ils ne devraient pas l'être
  - L'annulation d'un coup avec la flèche de gauche

Sources des images/sons : chess.com
